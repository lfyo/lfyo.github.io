    在武汉找工作，遇到太多的外包培训机构

    辛辛苦苦跑去面试，才发现是一家外包的培训机构，说的天花乱坠，就是贷款培训，包工作（是真的包还是假的，就说不清楚了）

    • 如果零技术，直接就会开门见山，让你去培训
    • 如果你的技术有那么一点点，就会问你有没有过项目经验，稍微的问一下技术层面的东西，最后就会显出原形：有没有想过培训
    • 如果技术挺好的，我们还有考虑考虑，等通知！！
    • 在武汉几百万的毕业生，还有这么多的挂着羊皮买狗肉的培训机构，很是难受～～～
    • 希望大家不要被坑，不要浪费自己的时间
    • 一般我都会在简历的后面备注：培训机构勿扰！！！（少了很多家培训公司打电话我）
    • 一般什么零基础的啊，包吃包住的、一带一的教…（自己去好好的判定～）
    • 会不定期的更新…
